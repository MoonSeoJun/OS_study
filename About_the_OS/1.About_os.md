# 운영체제란 무엇인가?
- 운영체제란 사용자에게 컴퓨터 하드웨어를 제어하는데 편의성을 제공하는 소프트웨어이다.

## 컴퓨터 부팅과정
![Boot]("C:\Users\user\Desktop\Boot.jpg")
1. 전원이 켜지면 CPU에서 ROM의 내용을 읽는다.  
2. POSt(Power-On Self-Test)는 전원이 켜지면 가장 먼저 실행되는 프로그램으로 컴퓨터의 상태를 확인한다.
3. Boot loader는 하드디스크에 저장 되어있는 운영체제를 찾아 RAM으로 보낸다.
4. 운영체제 실행

## 운영체제의 구조
- 운영체제는 크게 커널과 쉘로 나뉜다.

### 커널
- 운영체제의 핵심으로 운영체제가 수행하는 모든 것이 저장 되어있다.

### 쉘
- 사용자의 명령어를 해석하여 커널에 요청을 보내고 결과를 출력한다.
